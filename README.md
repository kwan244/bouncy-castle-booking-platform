# Bouncy Castle Booking Platform
A full-stack online booking system for a local bouncy castle rental business.

## Overview
This project is a modern full-stack web application that allows customers to browse bouncy castles, view details, and (later) submit booking requests.
It includes a professional development workflow with:
- GitHub Issues
- GitHub Project board
- Issue/PR templates
- Feature branching
- Planned sprints

---

### Tech Stack
- Frontend
- Next.js 14 (App Router)
- React
- TypeScript
- Tailwind CSS
- Backend
- Next.js API Routes (planned)
- MongoDB Atlas via Mongoose (planned)
- Other
- GitHub Projects (Agile workflow)
- GitHub Actions (planned CI)
- Stripe (planned payment integration)
- Resend / SendGrid (planned emails)
- Vercel (planned deployment)

### Setup & Installation
Clone the repository
git clone https://github.com/kwan244/bouncy-castle-booking-platform.git
cd bouncy-castle-booking-platform

Install dependencies
npm install

Run the development server
npm run dev


Visit:
http://localhost:3000

### Workflow

This project follows a professional development workflow:
1. Create an issue
2. Create a feature branch
3. git checkout -b feature/my-feature
4. Commit changes
5. Push and open a Pull Request
6. PR uses a structured template
7. PR closes the related issue
8. Merge into main
